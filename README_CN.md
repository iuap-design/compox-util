## 如何参与贡献


### 反馈

### 开发及构建

开发者可以一起参与为 compox-util 贡献代码，同时也可以基于 compox-util 进行二次开发或封装插件。


compox-util 使用 [gulp.js](http://gulpjs.com/) 构建项目。

克隆项目文件:

```
$ git clone git@github.com:iuap-design/compox-util.git
```

然后进入目录安装依赖：

```
$ npm install
```

接下来，执行 `gulp`：

```
$ npm run product
```
