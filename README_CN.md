##  介绍
`compox-util`主要提供createApp产出对象的工具类方法



## 安装

首先选择一个目录，克隆项目文件:

```
$ git clone git@github.com:iuap-design/compox-util.git
```



然后进入目录安装依赖：

```
$ cd compox-util && npm install
```



接下来，执行下面的命令产出：

```
$ npm run product
```
